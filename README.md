# Pavel's Dotfiles
My OS X dotfiles.

## Requirements
```
$ xcode-select --install
```

## Install
```
$ git clone https://github.com/facetheheat/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles && ./bootstrap.sh
$ sh brew.sh
$ sh cask.sh
$ sh osx.sh
$ sh applications.sh
```
